////文件结构////

assets 静态引用文件夹
-------------------- css 样式文件夹
---------------------------------------- app-default.css 项目初始公共合并后样式文件（尽量不要修改）
---------------------------------------- main.css 项目所需私有样式文件
-------------------- fonts 字体文件夹
---------------------------------------- bootstrap 图标字体
---------------------------------------- fa 图标字体
-------------------- images 图片文件夹
-------------------- js 项目所需功能私有文件价
---------------------------------------- app-webapp.js 项目私有功能（包括 项目配置 appConfig，项目js工具类 appUitl，项目公共初始化 appInit）
---------------------------------------- admin 文件夹（管理后台所用js文件夹）
-------------------------------------------------------------------------------- admin-common.js 管理后台所共有js方法
---------------------------------------- usercenter 文件夹（用户中心所用js文件夹）
-------------------------------------------------------------------------------- usercenter-common.js 个人中心所共有js方法
-------------------- plugins 插件文件夹
---------------------------------------- app-plugins.min.js 项目初始公共合并后插件（尽量不要修改）
---------------------------------------- 其他插件（公共性不强，最好是个别页面单独引用）
template 页面模板文件夹
-------------------- index.html 项目首页模板文件
-------------------- error.html 项目报错后文件
-------------------- admin 管理后台模板文件夹
-------------------- common 公共模板部分文件夹（模板公共部分）
---------------------------------------- head.html 页面公共头部部分文件 
---------------------------------------- foot.html 页面公共底部部分文件 
---------------------------------------- css.html 页面公共引用css部分文件 
---------------------------------------- script.html 页面公共引用js部分文件 
-------------------- usercenter 个人中心文件夹
test 测试文件夹
